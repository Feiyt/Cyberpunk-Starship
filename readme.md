# Cyberpunk Yellow-Green

[中文说明](readme-zh.md)

A cyberpunk yellow-green theme for [Starship](https://starship.rs/), inspired by [Gruvbox Rainbow](https://starship.rs/presets/gruvbox-rainbow).

## Preview

![Demo Preview](demo.gif)

## Features

- **Cyberpunk Style**: Distinctive yellow-green color palette.
- **Comprehensive Support**: Custom icons for OS and languages (Node.js, Python, Rust, Go, Java, PHP, etc.).
- **Cross-Platform**: Works on Windows, Linux, and macOS with PowerShell, Bash, Zsh, etc.

## Usage

1. **Install Starship**: Follow the [official guide](https://starship.rs/guide/#installation).
2. **Apply Theme**:
   Copy Cyberpunk Yellow-Green.toml to your Starship config location (usually ~/.config/starship.toml).
3. **Enable**:
   Add the init script to your shell configuration (e.g., Invoke-Expression (&starship init powershell) for PowerShell).
